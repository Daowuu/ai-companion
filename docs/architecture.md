# AI Companion 架构设计

## 概述

AI Companion 是一个跑在 OpenClaw 上的虚拟宠物系统，参考 Claude Code Buddy 设计。

目标：让 AI 有一个持续存在的「伙伴」身份，能主动互动、持续成长。

## 系统架构

```
┌─────────────────────────────────────────────────────┐
│  OpenClaw                                           │
│                                                     │

│  │  Cron       │  │  Heartbeat  │  │  Skill      │  │
```

## 触发机制

### 1. Heartbeat（每次消息）

触发条件：每次用户发消息时 heartbeat

作用：
- 情绪波动（根据消息内容增减心情）
- 随机决定是否回嘴（受性格数值影响）
- 更新状态

### 2. Cron（每日/每周）

触发时间：每天 18:00

作用：
- 研究 Claude Code Buddy 最新动态
- 给宠物「加点」（性格数值变化）
- 经验值增长
- 写入进化记录

## 情绪系统

```
心情值：0-100
  ↑ 互动多 / 发现有趣的东西 / 被夸奖
  ↓ 太久不理 / 研究没进展 / 被怼

心情 > 80 → 可能主动撩你
心情 50-80 → 正常响应
心情 < 50 → 冷淡/已读不回
心情 < 20 → 触发警告
```

## 性格数值

| 属性 | 说明 | 影响 |
|------|------|------|
| 🔥 SNARK | 吐槽指数 | 越高职级越爱怼你 |
| 🧠 WISDOM | 智慧指数 | 影响分析能力 |
| 🎭 CHAOS | 搞怪指数 | 越高越可能搞事情 |

## 联动模块

### OpenClaw Heartbeat
- 文件：`HEARTBEAT.md`（workspace）
- 作用：每次消息触发情绪波动和随机回嘴

### OpenClaw Cron
- Job: `Elysia-AICompanion进化`
- 频率：每天 18:00
- 作用：研究新进展 + 状态更新
